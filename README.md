# **Week 5 Assignment. Work Day Scheduler**

## **Description**
### In the fifth week assignment the students had a task to Create a simple calendar application that allows a user to save events for each hour of the day by modifying starter code. This app will run in the browser and feature dynamically updated HTML and CSS powered by jQuery.
